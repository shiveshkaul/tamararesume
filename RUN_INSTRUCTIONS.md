# TamaraApply Pro - Run Instructions

To run the application locally, follow these steps:

## Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your system.

## 1. Start the Backend
Open a new terminal and run:
```bash
cd "/Users/shiveshkaul/Desktop/Jarvis Jobs/tamaraapply-pro/backend"
npx ts-node src/index.ts
```
> The backend will be available at: **http://localhost:8000**

## 2. Start the Frontend
Open another terminal and run:
```bash
cd "/Users/shiveshkaul/Desktop/Jarvis Jobs/tamaraapply-pro/frontend"
npm run dev
```
> The frontend will be available at: **http://localhost:3000**

## 3. Use the App
1. Open your browser and navigate to **http://localhost:3000**.
2. Paste a German Job Description.
3. Click "Tailor with AI" to generate your tailored resume and cover letter.
